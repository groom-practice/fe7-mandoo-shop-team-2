# Mandooo Shop

## 🛍️ 프로젝트 소개

'Mandooo Shop'은 다양한 만두 관련 굿즈를 판매하는 온라인 쇼핑몰입니다. 이 프로젝트는 프론트엔드 개발 학습을 위해 만들어진 웹사이트입니다.

## ✨ 주요 기능

- **메인 페이지**: 쇼핑몰의 첫인상을 주는 페이지입니다.
- **로그인/회원가입**: 사용자가 계정을 만들고 로그인할 수 있습니다.
- **상품 목록 페이지**: 판매하는 모든 상품을 둘러볼 수 있습니다.
- **장바구니 페이지**: 원하는 상품을 담고, 수량을 조절하거나 삭제할 수 있습니다.
- **주문 페이지**: 장바구니에 담은 상품을 주문할 수 있습니다.

## 🛠️ 기술 스택

- HTML
- CSS
- JavaScript
